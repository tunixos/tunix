# Tunix

Tunix is a small and simple Unix-like operating system experiment for x86_64.

<img width="1332" height="836" alt="image" src="https://github.com/user-attachments/assets/41174df4-e611-49d8-93de-04e438950905" />

## Features

- Custom bootloader and kernel code
- Basic userspace tools
- BusyBox, Bash, and TinyCC support

## Build

```sh
make all
```

## Run

```sh
make run
```

To run headless:

```sh
make headless
```

## Clean

```sh
make clean
```
